## 支持的模型

启航 AI 提供多种类型的 AI 模型:

- **文本聊天**: GPT-5, GPT-4.1, Claude Opus 4, DeepSeek R1, Kimi K2 等
- **图像生成**: 启航绘图 X2, DALL-E 3, Nano Banana 2, 即梦 4.0 等
- **语音合成**: 启航 TTS (38+ 语音角色)
- **视频生成**: Sora 2, Sora 2 HD
- **音乐生成**: Suno V4.5 Plus, Suno V5
- **嵌入模型**: BGE-M3, Text Embedding 3 Large
- **重排序**: BGE Reranker V2 M3

## 快速示例

<CodeGroup>

```python Python
import openai

client = openai.OpenAI(
    api_key="sk-your-api-key-here",
    base_url="https://api.qhaigc.net/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "user", "content": "你好,介绍一下启航 AI"}
    ]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key-here',
  baseURL: 'https://api.qhaigc.net/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [
    { role: 'user', content: '你好,介绍一下启航 AI' }
  ]
});

console.log(response.choices[0].message.content);
```

</CodeGroup>

## 获取 API Key

1. 访问 [启航 AI 控制台](https://api.qhaigc.net/v1)
2. 注册并登录您的账号
3. 在 **API 令牌** 页面生成您的 API Key
4. 开始使用!

## 欢迎使用启航 AI API

启航 AI API 系统是一款聚焦低成本探索 AIGC 无限可能的服务平台,提供文本聊天、图像生成、语音合成、视频生成等多种 AI 能力。

## 获取 API Key

1. 访问 [启航 AI 控制台](https://api.qhaigc.net/v1)
2. 注册并登录您的账号
3. 在 **API 令牌** 页面生成您的 API Key
4. 妥善保管您的 API Key,不要泄露给他人

<Warning>
  API Key 具有完全的账户权限,请务必保密。如果不慎泄露,请立即在控制台重新生成。
</Warning>

## 快速调用示例

### 文本聊天

<CodeGroup>

```python Python
import openai

client = openai.OpenAI(
    api_key="sk-your-api-key-here",
    base_url="https://api.qhaigc.net/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[
        {"role": "user", "content": "你好,介绍一下启航 AI"}
    ]
)

print(response.choices[0].message.content)
```

```javascript JavaScript
import OpenAI from 'openai';

const client = new OpenAI({
  apiKey: 'sk-your-api-key-here',
  baseURL: 'https://api.qhaigc.net/v1'
});

const response = await client.chat.completions.create({
  model: 'gpt-4o',
  messages: [
    { role: 'user', content: '你好,介绍一下启航 AI' }
  ]
});

console.log(response.choices[0].message.content);
```

```bash cURL
curl https://api.qhaigc.net/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer sk-your-api-key-here" \
  -d '{
    "model": "gpt-4o",
    "messages": [
      {"role": "user", "content": "你好,介绍一下启航 AI"}
    ]
  }'
```

</CodeGroup>

## 常见问题

<AccordionGroup>
  <Accordion title="如何处理 API 错误?">
    启航 AI API 使用标准的 HTTP 状态码。常见错误包括:
    - `401 Unauthorized`: API Key 无效或未提供
    - `429 Too Many Requests`: 请求频率超限
    - `500 Internal Server Error`: 服务器错误
    
    建议在代码中添加错误处理和重试逻辑。
  </Accordion>
  
  <Accordion title="支持哪些编程语言?">
    启航 AI API 兼容 OpenAI 接口标准,支持所有主流编程语言:
    - Python (推荐使用 `openai` 库)
    - JavaScript/TypeScript (推荐使用 `openai` npm 包)
    - Java, Go, PHP, Ruby 等
    
    也可以使用标准的 HTTP 客户端直接调用。
  </Accordion>
  
  <Accordion title="如何控制成本?">
    - 选择合适的模型(不同模型价格不同)
    - 设置 `max_tokens` 限制输出长度
    - 使用流式响应及时中断不需要的内容
    - 在控制台设置预算提醒
  </Accordion>
</AccordionGroup>

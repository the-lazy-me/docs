## 接口/方法列表

以下是一些常用的接口标识或 SDK 方法名：

| 标识 | 说明 |
| :--- | :--- |
| `eventHistory` | 历史事件 |
| `eventHistoryGet` | 获取历史事件 |
| `eventHistorySearch` | 搜索历史事件 |
| `musicSearch` | 音乐搜索 |
| `musicDetail` | 音乐详情 |
| `musicUrl` | 音乐链接 |
| `musicLyric` | 音乐歌词 |
| `musicCommentHot` | 音乐热评 |
| `musicPlaylist` | 歌单信息 |
| `tianqi` | 天气信息 |
| `tianqiSeven` | 七天天气 |
| `wbtop` | 微博热搜 |
| `zaobao` | 早报 |
| `zhihu` | 知乎热榜 |
| `zhihuGet` | 获取知乎内容 |
| `zhihuNews` | 知乎新闻 |
| `zhihuShortComments` | 知乎短评 |

> **注意**：具体参数请参考各模块详细文档。

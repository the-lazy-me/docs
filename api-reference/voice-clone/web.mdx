## 在线创建

您可以直接访问我们的在线实验室进行语音角色的创建和管理。

**访问地址**：[https://lab.qhaigc.net](https://lab.qhaigc.net)

## 使用步骤

1. 登录 [在线实验室](https://lab.qhaigc.net)。
2. 点击“创建语音模型”。
3. 填写模型名称和描述（可选）。
4. 选择是否公开。
5. 上传参考音频并开始训练。

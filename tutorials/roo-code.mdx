# Roo Code 接入启航AI教程

[Roo Code](https://github.com/RooVetGit/Roo-Code) 是 VS Code AI 编程插件，Cline 的分支版本。本教程介绍如何接入启航AI。

## 前置要求

- VS Code
- Roo Code 插件
- 启航AI API Key（[获取地址](https://api.qhaigc.net)）

## 安装 Roo Code

### 从 VS Code 扩展市场安装

1. 打开 VS Code
2. 点击扩展图标（`Ctrl/Cmd + Shift + X`）
3. 搜索 **Roo Code**
4. 点击 **安装**

## 配置步骤

### 1. 打开 Roo Code 面板

安装完成后，点击左侧 Roo Code 图标。

### 2. 进入设置

点击面板右上角的 **设置** 图标（齿轮）。

### 3. 选择 API Provider

在 **API Provider** 下拉框中选择 **OpenAI Compatible**。

### 4. 填写配置

| 配置项 | 值 |
|--------|-----|
| API Key | 你的启航AI API Key |
| Base URL | `https://api.qhaigc.net/v1` |
| Model | 选择模型，如 `gpt-4o` |

### 5. 选择模型

在 **Model** 下拉框中选择启航AI提供的模型：

- `gpt-4o`
- `gpt-4o-mini`
- `claude-3-5-sonnet`
- `claude-3-haiku`

### 6. 保存并开始使用

关闭设置面板，即可开始使用。

## 与 Cline 的区别

Roo Code 是 Cline 的分支，主要区别：

- ✅ 更多自定义选项
- ✅ 更好的代码审查功能
- ✅ 支持更多模型参数调整
- ✅ 改进的 UI 界面

在设置中可以自定义系统提示词：


可以在对话中快速切换模型：

- **复杂任务**：claude-3-5-sonnet
- **快速原型**：gpt-4o-mini
- **综合开发**：gpt-4o

### 代码片段管理

Roo Code 支持保存常用代码片段：

1. 选中代码
2. 点击 **保存片段**
3. 在后续对话中引用

## 常见问题

**Q: 与 Cline 冲突？**

A: 建议只安装其中一个插件，避免快捷键冲突。

**Q: 连接失败？**

A: 检查：
- Base URL 是否包含 `/v1`
- API Key 是否正确
- 网络连接是否正常

**Q: 如何查看 Token 消耗？**

A: 登录 [启航AI控制台](https://api.qhaigc.net) 查看。

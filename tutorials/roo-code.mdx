# Roo Code 接入启航AI教程

## 教程说明

本教程介绍如何在 Roo Code 中接入启航AI。Roo Code 支持 OpenAI-compatible 接口，但公开资料对部分 UI 字段的描述不够统一，因此本文只写可确认的配置方向。

## 适用版本或适用形态

- VS Code 中安装的 Roo Code 扩展
- 能在设置中选择 provider 的 Roo Code 版本

## 前置条件

- VS Code
- 已安装 Roo Code
- 启航AI API Key

## 官方支持的接入方式

Roo Code 可在 provider 设置中使用 OpenAI-compatible 接口。可确认的配置项包括：

- `API Provider`
- `Base URL`
- `API Key`
- `Model`

## 逐步配置

1. 打开 Roo Code 面板。
2. 进入设置页面。
3. 选择 OpenAI-compatible 相关 provider。
4. 填写启航AI的 `Base URL` 与 `API Key`。
5. 选择或手动填写要使用的模型名。
6. 保存配置后重新打开会话。

## 验证是否接入成功

- Roo Code 可以正常保存配置
- 面板中可以发起对话
- 返回内容来自启航AI，而不是报认证或连接错误

## 常见问题

**Q: 具体字段名和入口和 Cline 一样吗？**

A: 不强行假设完全一致。只按 Roo Code 当前可见的 provider 设置填写，字段名以实际 UI 为准。

**Q: 模型为什么没有自动显示？**

A: 先按 `Base URL` 和 `API Key` 验证连接，再手动填写模型名。

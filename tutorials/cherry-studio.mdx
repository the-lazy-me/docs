# Cherry Studio 接入启航AI教程

[Cherry Studio](https://cherry-ai.com) 是国产多模型 AI 桌面客户端，本土化强，增长快。本教程介绍如何接入启航AI。

## 前置要求

- Cherry Studio 客户端（[下载地址](https://cherry-ai.com)）
- 启航AI API Key（[获取地址](https://api.qhaigc.net)）

## 配置步骤

### 1. 打开设置

点击右上角 **设置** 图标（齿轮）。

### 2. 进入模型服务

在设置页面中选择 **模型服务** 或 **Model Service**。

### 3. 添加自定义服务商

点击 **添加** 或 **+** 按钮，选择 **OpenAI 兼容**。

### 4. 填写启航AI配置

| 配置项 | 值 |
|--------|-----|
| 名称 | 启航AI |
| API 地址 | `https://api.qhaigc.net/v1` |
| API Key | 你的启航AI API Key |
| 模型 | 可选填，留空则自动获取 |

### 5. 保存并测试

点击 **保存**，然后点击 **测试连接** 验证配置。

### 6. 选择模型

在对话页面，点击模型选择下拉框，选择启航AI提供的模型。

## 支持的模型

Cherry Studio 支持启航AI的所有模型：

| 模型 | 特点 |
|------|------|
| gpt-4o | 最强多模态 |
| gpt-4o-mini | 高性价比 |
| claude-3-5-sonnet | 最强推理 |
| claude-3-haiku | 快速响应 |
| gemini-1.5-pro | Google模型 |

## 高级功能

### 助手配置

可以创建不同助手，绑定不同模型：

1. 点击左侧 **助手**
2. 点击 **创建助手**
3. 选择启航AI的特定模型
4. 设置系统提示词

### 知识库

Cherry Studio 支持本地知识库：

1. 点击 **知识库**
2. 上传文档
3. 在对话时选择知识库

### 多模型对比

可以同时开启多个对话窗口，使用不同模型对比回答。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + N` | 新建对话 |
| `Ctrl/Cmd + Shift + N` | 新建助手 |
| `Ctrl/Cmd + ,` | 打开设置 |

## 常见问题

**Q: 连接测试失败？**

A: 检查 API 地址是否包含 `/v1`，API Key 是否正确。

**Q: 模型列表为空？**

A: 点击刷新按钮，或手动输入模型名称。

**Q: 如何更新模型列表？**

A: 在模型服务设置中点击 **刷新模型列表**。

# 启航AI 教程索引

## 说明

本页只汇总两类内容：

- 已按官方文档或官方仓库核对过的接入教程
- 尚未补齐字段级证据、因此暂缓更新的高风险页面

如果某个平台当前官方资料不足，或暂时无法确认其是否支持 OpenAI-compatible 自定义地址，本索引不会把它继续放在“可直接接入”区域。

## 通用前提

大多数教程都会用到以下信息：

- 启航AI API Key：[前往控制台创建](https://www.qhaigc.net/console/apikeys)
- OpenAI 兼容地址：`https://api.qhaigc.net/v1`

<Info>
  是否需要填写 <code>/v1</code> 取决于平台自身字段定义。正文中若没有写死，表示该平台的官方资料没有给出绝对规则。
</Info>

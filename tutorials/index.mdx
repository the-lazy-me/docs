# 启航AI 生态接入文档索引

欢迎来到启航AI生态接入文档中心。本文档汇总了启航AI与各大主流平台的接入教程。

## 🚀 快速开始

所有平台接入都需要：

1. **获取 API Key**：[https://api.qhaigc.net](https://api.qhaigc.net)
2. **配置 Base URL**：`https://api.qhaigc.net/v1`
3. **选择模型**：gpt-4o / claude-3-5-sonnet 等

## 📈 接入优先级建议

### 第一阶段：核心平台（立即接入）

1. **Dify** - 国内最大LLM应用平台
2. **New API** - B端渠道商，批量带来用户
3. **n8n** - 自动化场景，token消耗高
4. **AnythingLLM** - 知识库场景高频使用
5. **LobeChat/Cherry Studio/Chatbox** - 客户端三巨头
6. **Cline/Continue/Claude Code** - 开发者工具

### 第二阶段：快速跟进

- Roo Code / Kilo Code
- Flowise / Langflow
- SillyTavern
- AstrBot
- RAGFlow

### 第三阶段：持续布局

- OpenCode
- Monkey Code
- LangBot

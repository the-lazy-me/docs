# Chatbox 接入启航AI教程

[Chatbox](https://chatboxai.com) 是国内最早流行的跨平台 AI 桌面客户端，用户基数大。本教程介绍如何接入启航AI。

## 前置要求

- Chatbox 客户端（[下载地址](https://chatboxai.com)）
- 启航AI API Key（[获取地址](https://api.qhaigc.net)）

## 配置步骤

### 1. 打开设置

点击左下角 **设置** 图标，或使用快捷键 `Ctrl/Cmd + ,`。

### 2. 选择AI服务商

在设置页面的 **AI 服务商** 或 **Provider** 选项中，选择 **OpenAI**。

### 3. 填写配置

| 配置项 | 值 |
|--------|-----|
| API 域名 | `https://api.qhaigc.net` |
| API Key | 你的启航AI API Key |

> ⚠️ 注意：Chatbox 的 API 域名不需要填写 `/v1`，只需填写到域名级别。

### 4. 选择模型

在模型下拉框中选择启航AI提供的模型：

- `gpt-4o`
- `gpt-4o-mini`
- `claude-3-5-sonnet`
- `claude-3-haiku`
- `gemini-1.5-pro`

### 5. 保存设置

点击 **保存** 或 **Apply**，配置生效。

- ✅ 多模型切换

## 使用技巧

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl/Cmd + N` | 新对话 |
| `Ctrl/Cmd + Shift + O` | 切换模型 |
| `Ctrl/Cmd + /` | 打开命令面板 |

### 提示词模板

可以在设置中创建常用提示词模板：

1. 点击 **模板** 或 **Prompts**
2. 点击 **添加模板**
3. 设置名称和提示词内容
4. 在对话中快速调用

## 常见问题

**Q: API 域名应该填什么？**

A: 只填域名，不包含路径：`https://api.qhaigc.net`

**Q: 连接失败？**

A: 检查：
- API Key 是否正确
- 网络能否访问 `api.qhaigc.net`
- 是否选择了正确的服务商类型

**Q: 模型列表为空？**

A: Chatbox 可能会自动获取模型列表，如未显示，可手动输入模型名称。

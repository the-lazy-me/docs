# Chatbox 接入启航AI教程

## 教程说明

Chatbox 通过 `OpenAI` provider 接入启航AI。官方当前提供的是 `API Host` 和 API Key 这类字段，而不是一套固定的模型网关模板。

## 适用版本或适用形态

- Chatbox 桌面端
- 支持自定义 API Host 的版本

## 前置条件

- Chatbox 客户端
- 启航AI API Key
- 启航AI可访问的 API Host

## 官方支持的接入方式

- 在 `Settings` 中选择 `OpenAI` provider
- 填写 `API Host`
- 填写 API Key
- 由上游服务决定是否可见模型列表

## 逐步配置

### 1. 打开设置

1. 启动 Chatbox
2. 打开设置页

### 2. 选择 provider

1. 在 `AI Provider` 或 `Provider` 中选择 `OpenAI`
2. 进入对应配置页

### 3. 填写启航AI信息

1. 填写 `API Host`
2. 填写 `API Key`
3. 保存设置

### 4. 开始使用

1. 新建对话
2. 选择可用模型
3. 开始测试

## 验证是否接入成功

- 发送一条测试消息
- 如果能收到回复，说明连接成功
- 如果模型列表为空，先确认上游和 API Host 配置是否正确

## 常见问题

**Q: API Host 应该填到什么层级？**

A: 按 Chatbox 当前版本的 `API Host` 字段填写；如果你的上游网关有自己的路径规范，以网关规范为准，不要写到具体的 completions 路径。

**Q: 连接失败怎么办？**

A: 检查 API Key、API Host 和网络可达性。

**Q: 为什么没有自动显示模型？**

A: 模型可见性取决于上游返回和当前客户端版本。

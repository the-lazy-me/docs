# Monkey Code 接入启航AI教程

[Monkey Code](https://github.com/monkey-code-ai/monkey-code) 是国产 AI 编程工具，国内开发者用户。本教程介绍如何接入启航AI。

## 前置要求

- VS Code
- Monkey Code 插件
- 启航AI API Key（[获取地址](https://api.qhaigc.net)）

## 安装 Monkey Code

### 从 VS Code 扩展市场安装

1. 打开 VS Code
2. 点击扩展图标（`Ctrl/Cmd + Shift + X`）
3. 搜索 **Monkey Code**
4. 点击 **安装**

### 从 GitHub 安装

```bash
# 克隆仓库
git clone https://github.com/monkey-code-ai/monkey-code.git
cd monkey-code

# 安装依赖
npm install

# 打包
npm run package

# 在 VS Code 中加载 .vsix 文件
code --install-extension monkey-code-*.vsix
```

## 配置步骤

### 1. 打开 Monkey Code 面板

安装完成后，点击左侧 Monkey Code 图标。

### 2. 进入设置

点击面板右上角的 **设置** 图标（齿轮）。

### 3. 选择 API 类型

在 **API 类型** 下拉框中选择 **OpenAI 兼容**。

### 4. 填写配置

| 配置项 | 值 |
|--------|-----|
| API Key | 你的启航AI API Key |
| Base URL | `https://api.qhaigc.net/v1` |
| 模型 | 选择模型，如 `gpt-4o` |

### 5. 选择模型

在 **模型** 下拉框中选择启航AI提供的模型：

- `gpt-4o`
- `gpt-4o-mini`
- `claude-3-5-sonnet`
- `claude-3-haiku`

### 6. 保存并开始使用

关闭设置面板，即可开始使用。

## 常见问题

**Q: 连接失败？**

A: 检查：
- Base URL 是否包含 `/v1`
- API Key 是否正确
- 网络连接是否正常

**Q: 如何查看 Token 消耗？**

A: 登录 [启航AI控制台](https://api.qhaigc.net) 查看。

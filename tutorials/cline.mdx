# Cline 接入启航AI教程

## 教程说明

本教程介绍如何在 Cline 中接入启航AI。Cline 官方支持 `OpenAI Compatible` provider，因此可以按兼容 OpenAI 的方式填写启航AI接口信息。

## 适用版本或适用形态

- VS Code 中安装的 Cline 扩展
- Cline 面板中可进入 provider 设置的版本

## 前置条件

- VS Code
- 已安装 Cline
- 启航AI API Key

## 官方支持的接入方式

在 Cline 的 provider 设置中选择 `OpenAI Compatible`，然后填写兼容接口所需信息：

- `API Provider`: `OpenAI Compatible`
- `Base URL`
- `API Key`
- `Model ID`

## 逐步配置

1. 打开 VS Code 中的 Cline 面板。
2. 点击设置入口，进入 provider 配置。
3. 将 `API Provider` 选为 `OpenAI Compatible`。
4. 填入启航AI的 `Base URL` 和 `API Key`。
5. 填入要使用的 `Model ID`，例如你在启航AI控制台中可用的模型名。
6. 使用 Cline 的校验功能验证配置。

## 验证是否接入成功

- 能够在 Cline 面板中正常发起对话或任务
- 校验配置时不再报认证错误
- 发送一个简单指令后，Cline 能返回启航AI的模型回复

## 常见问题

**Q: 连接失败怎么办？**

A: 先检查 `API Key`、`Base URL` 和 `Model ID` 是否填写正确，再确认网络可以访问启航AI接口。

**Q: 模型列表没有自动出现怎么办？**

A: 直接按 Cline 的 provider 设置手动填写 `Model ID`。不要依赖未确认的自动拉取行为。

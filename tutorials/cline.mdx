# Cline 接入启航AI教程

[Cline](https://github.com/cline/cline) 是 VS Code AI 编程插件，支持自主完成复杂编程任务。本教程介绍如何接入启航AI。

## 前置要求

- VS Code
- Cline 插件（从 VS Code 扩展市场安装）
- 启航AI API Key（[获取地址](https://api.qhaigc.net)）

## 安装 Cline

### 从 VS Code 扩展市场安装

1. 打开 VS Code
2. 点击左侧扩展图标（或按 `Ctrl/Cmd + Shift + X`）
3. 搜索 **Cline**
4. 点击 **安装**

## 配置步骤

### 1. 打开 Cline 面板

安装完成后，点击左侧 Cline 图标打开面板。

### 2. 进入设置

点击面板右上角的 **设置** 图标（齿轮）。

### 3. 选择 API Provider

在 **API Provider** 下拉框中选择：

- **OpenAI Compatible**（推荐）
- 或 **OpenAI**

### 4. 填写配置

| 配置项 | 值 |
|--------|-----|
| API Key | 你的启航AI API Key |
| Base URL | `https://api.qhaigc.net/v1` |
| Model | 选择模型，如 `gpt-4o` |

### 5. 选择模型

在 **Model** 下拉框中选择启航AI提供的模型：

- `gpt-4o` - 最强编程能力
- `gpt-4o-mini` - 快速响应
- `claude-3-5-sonnet` - 推荐用于复杂编程
- `claude-3-haiku` - 快速简单任务

### 6. 保存并开始使用

关闭设置面板，即可开始使用 Cline。

## 使用方式

### 自然语言编程

在 Cline 面板中输入自然语言描述：

```
创建一个 React 组件，包含一个表单，可以添加和删除待办事项
```

Cline 会自动：
1. 分析需求
2. 创建/修改文件
3. 编写代码
4. 运行命令

### 代码解释

选中代码，输入：

```
解释这段代码的作用
```

### 代码优化

选中代码，输入：

```
优化这段代码的性能
```

### 修复 Bug

输入：

```
修复这个文件中的错误
```

## 常见问题

**Q: 无法连接 API？**

A: 检查：
- Base URL 是否包含 `/v1`
- API Key 是否正确
- 网络能否访问 `api.qhaigc.net`

**Q: 模型列表为空？**

A: 手动输入模型名称，如 `gpt-4o` 或 `claude-3-5-sonnet`。

**Q: 如何查看 Token 消耗？**

A: 登录 [启航AI控制台](https://api.qhaigc.net) 查看。

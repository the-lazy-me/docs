# SillyTavern 接入启航AI教程

[SillyTavern](https://sillytavern.com) 是 AI 角色扮演/创意写作平台，用户粘性极高，token 消耗量大。本教程介绍如何接入启航AI。

## 前置要求

- SillyTavern（[下载地址](https://sillytavern.com)）
- 启航AI API Key（[获取地址](https://api.qhaigc.net)）

## 配置步骤

### 1. 打开设置

点击左下角 **_settings_（设置）图标，或使用快捷键 `Ctrl/Cmd + ,`。

### 2. 进入 API 设置

在设置页面中找到 **API** 或 **API Configuration**。

### 3. 选择 API 类型

在 **API Type** 或 **API** 下拉框中选择：

- **OpenAI** 或 **OpenAI Compatible**
- **Claude**（如需使用 Claude 模型）

### 4. 填写配置

#### OpenAI 兼容配置：

| 配置项 | 值 |
|--------|-----|
| API Base URL | `https://api.qhaigc.net/v1` |
| API Key | 你的启航AI API Key |

#### Claude 配置（如使用 Claude 模型）：

| 配置项 | 值 |
|--------|-----|
| API Type | Claude (Anthropic) |
| API Base URL | `https://api.qhaigc.net/v1` |
| API Key | 你的启航AI API Key |

### 5. 选择模型

在 **Model** 或 **模型** 下拉框中选择：

- `gpt-4o` - 推荐用于角色扮演
- `gpt-4o-mini` - 快速响应
- `claude-3-5-sonnet` - 最适合角色扮演
- `claude-3-haiku` - 快速响应

### 6. 保存并测试

点击 **Save** 保存设置，然后发送测试消息验证连接。

## 高级配置

### 自定义端点

如果启航AI提供专用端点，可在 **Advanced** 中设置：

```
https://api.qhaigc.net/v1/chat/completions
```

### 连接参数

可在设置中调整：

| 参数 | 建议值 | 说明 |
|------|--------|------|
| Context Length | 8192 | 上下文长度 |
| Max Tokens | 2048 | 单次响应上限 |
| Temperature | 0.8 | 创造性 |

### 多模型配置

可以配置多个 API 提供商，在不同场景切换：

1. 创建多个 API 配置
2. 在对话中快速切换

## 常见问题

**Q: 连接失败？**

A: 检查：
- API Base URL 是否包含 `/v1`
- API Key 是否正确
- 网络能否访问 `api.qhaigc.net`

**Q: Claude 模型不可用？**

A: 确保在 Claude API 类型下配置，或使用 OpenAI 兼容模式调用 Claude 模型。

**Q: 如何减少 Token 消耗？**

A: 在设置中降低 Context Length，或使用更短的角色设定。

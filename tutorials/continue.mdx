# Continue 接入启航AI教程

[Continue](https://continue.dev) 是 VS Code 和 JetBrains 的 AI 编程助手，支持代码补全和对话。本教程介绍如何接入启航AI。

## 前置要求

- VS Code 或 JetBrains IDE
- Continue 插件
- 启航AI API Key（[获取地址](https://api.qhaigc.net)）

## 安装 Continue

### VS Code

1. 打开 VS Code
2. 点击扩展图标（`Ctrl/Cmd + Shift + X`）
3. 搜索 **Continue**
4. 点击 **安装**

### JetBrains

1. 打开 JetBrains IDE
2. 进入 **Settings** → **Plugins**
3. 搜索 **Continue**
4. 点击 **Install**

## 配置步骤

### 1. 打开 Continue 面板

- VS Code: 点击左侧 Continue 图标或按 `Ctrl/Cmd + L`
- JetBrains: 点击右侧工具栏 Continue 图标

### 2. 打开配置文件

点击面板右上角的 **设置** 图标，选择 **Open config.json**。

### 3. 修改配置

在 `config.json` 中添加或修改以下配置：

```json
{
  "models": [
    {
      "title": "启航AI - GPT-4o",
      "provider": "openai",
      "model": "gpt-4o",
      "apiKey": "your-api-key",
      "apiBase": "https://api.qhaigc.net/v1"
    },
    {
      "title": "启航AI - Claude",
      "provider": "openai",
      "model": "claude-3-5-sonnet",
      "apiKey": "your-api-key",
      "apiBase": "https://api.qhaigc.net/v1"
    }
  ]
}
```

### 4. 选择模型

在 Continue 面板顶部的模型下拉框中，选择已配置的启航AI模型。

## 支持的模型

Continue 支持启航AI的所有模型：

| 模型 | 配置值 | 用途 |
|------|--------|------|
| GPT-4o | `gpt-4o` | 综合编程 |
| GPT-4o-mini | `gpt-4o-mini` | 快速响应 |
| Claude 3.5 Sonnet | `claude-3-5-sonnet` | 复杂任务 |
| Claude 3 Haiku | `claude-3-haiku` | 快速简单 |

## 使用方式

### 代码补全

Continue 会自动提供代码补全建议，按 `Tab` 接受。

### 内联编辑

选中代码，按 `Ctrl/Cmd + I`，输入指令：

```
重构这段代码，使其更易读
```

### 侧边栏对话

按 `Ctrl/Cmd + L` 打开侧边栏，进行对话：

```
解释这个函数的作用
```

### 错误诊断

当出现错误时，Continue 会自动提示解释和修复建议。

## 常见问题

**Q: 配置后无法连接？**

A: 检查：
- `apiBase` 是否包含 `/v1`
- `apiKey` 是否正确
- JSON 格式是否正确

**Q: JetBrains 中不显示？**

A: 重启 IDE 或重新安装插件。

**Q: 如何查看用量？**

A: 登录 [启航AI控制台](https://api.qhaigc.net) 查看。

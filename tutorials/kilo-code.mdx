# Kilo Code 接入启航AI教程

## 教程说明

本教程介绍如何在 Kilo Code 中接入启航AI。Kilo Code 官方支持 OpenAI Compatible provider，也支持先使用 Kilo 自身的登录/托管流程；本文只覆盖兼容接口接法。

## 适用版本或适用形态

- VS Code 中的 Kilo Code 扩展
- 支持 provider 选择的 Kilo Code 版本

## 前置条件

- VS Code
- 已安装 Kilo Code
- 启航AI API Key

## 官方支持的接入方式

在 provider 配置中选择 `OpenAI Compatible`，再填写：

- `API Provider`
- `Base URL`
- `API Key`
- `Model`

## 逐步配置

1. 打开 Kilo Code 面板。
2. 进入设置。
3. 选择 `OpenAI Compatible` provider。
4. 填写启航AI的 `Base URL` 和 `API Key`。
5. 选择或填写模型名。
6. 保存后重新发起一次对话。

## 验证是否接入成功

- 配置可以保存
- 代码对话能正常返回结果
- 没有出现 provider 认证失败或模型不可用错误

## 常见问题

**Q: 为什么不写团队共享配置或审计日志？**

A: 那些属于 Kilo Code 额外能力，不是启航AI接入的必要步骤。

**Q: 连接失败怎么办？**

A: 依次检查 `API Provider`、`Base URL`、`API Key` 和 `Model`，再确认网络连通性。

# OpenCode 接入启航AI教程

[OpenCode](https://github.com/opencode-ai/opencode) 是终端 AI 编程助手，类 Claude Code，开源。本教程介绍如何接入启航AI。

## 前置要求

- Node.js 18+
- OpenCode CLI
- 启航AI API Key（[获取地址](https://api.qhaigc.net)）

## 安装 OpenCode

```bash
npm install -g @opencode-ai/opencode
```

## 配置步骤

### 1. 设置环境变量

#### macOS/Linux

```bash
export OPENAI_API_KEY="your-api-key"
export OPENAI_BASE_URL="https://api.qhaigc.net/v1"
```

添加到 `~/.zshrc` 或 `~/.bashrc`：

```bash
echo 'export OPENAI_API_KEY="your-api-key"' >> ~/.zshrc
echo 'export OPENAI_BASE_URL="https://api.qhaigc.net/v1"' >> ~/.zshrc
source ~/.zshrc
```

#### Windows (PowerShell)

```powershell
$env:OPENAI_API_KEY = "your-api-key"
$env:OPENAI_BASE_URL = "https://api.qhaigc.net/v1"
```

### 2. 验证配置

```bash
opencode --version
```

### 3. 启动 OpenCode

在项目目录中：

```bash
opencode
```

## 支持的模型

OpenCode 支持启航AI的所有模型：

| 模型 | 配置方式 |
|------|---------|
| gpt-4o | 默认 |
| gpt-4o-mini | `--model gpt-4o-mini` |
| claude-3-5-sonnet | `--model claude-3-5-sonnet` |
| claude-3-haiku | `--model claude-3-haiku` |

## 高级用法

### 指定模型

```bash
opencode --model claude-3-5-sonnet
```

### 配置文件

创建 `~/.opencode/config.json`：

```json
{
  "apiKey": "your-api-key",
  "baseUrl": "https://api.qhaigc.net/v1",
  "defaultModel": "gpt-4o",
  "maxTokens": 4096
}
```

### 项目级配置

在项目根目录创建 `.opencode.json`：

```json
{
  "model": "claude-3-5-sonnet",
  "systemPrompt": "你是一个专业的 Python 开发者"
}
```

### 禁用工具

```bash
opencode --no-tools
```

### 仅对话模式

```bash
opencode --chat
```

## 常见问题

**Q: 提示 "API key not found"？**

A: 确保环境变量已正确设置，或检查配置文件。

**Q: 连接失败？**

A: 检查：
- `OPENAI_BASE_URL` 是否包含 `/v1`
- API Key 是否有效
- 网络连接是否正常

**Q: 如何查看 Token 消耗？**

A: 登录 [启航AI控制台](https://api.qhaigc.net) 查看。

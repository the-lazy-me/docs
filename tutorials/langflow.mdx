# Langflow 接入启航AI教程

[Langflow](https://langflow.org) 是可视化 LangChain 工作流构建工具。本教程介绍如何接入启航AI。

## 前置要求

- Langflow（pip 或 Docker 安装）
- 启航AI API Key（[获取地址](https://api.qhaigc.net)）

## 安装 Langflow

### pip 安装

```bash
pip install langflow
langflow run
```

### Docker 安装

```bash
docker run -d -p 7860:7860 \
  -e OPENAI_API_KEY=your-api-key \
  langflowai/langflow:latest
```

## 配置步骤

### 1. 打开 Langflow

访问 `http://localhost:7860`。

### 2. 创建新项目

点击 **New Project** 或 **+**。

### 3. 添加 OpenAI 组件

从左侧组件库中，拖入 **OpenAI** 组件到画布。

### 4. 配置组件

点击组件，在右侧配置面板中设置：

| 配置项 | 值 |
|--------|-----|
| API Key | 你的启航AI API Key |
| Base URL | `https://api.qhaigc.net/v1` |
| Model | `gpt-4o` 或其他模型 |
| Temperature | 0.7 |
| Max Tokens | 2048 |

### 5. 连接工作流

连接输入组件（如 Prompt）和输出组件（如 Chat Output）。

### 6. 运行测试

点击 **Run** 或 **Play** 按钮测试工作流。

## 支持的模型

Langflow 支持启航AI的所有模型：

| 模型 | 配置值 |
|------|--------|
| GPT-4o | `gpt-4o` |
| GPT-4o-mini | `gpt-4o-mini` |
| Claude 3.5 Sonnet | `claude-3-5-sonnet` |
| Claude 3 Haiku | `claude-3-haiku` |

## 常用工作流

### 简单对话

```
[Chat Input] → [Prompt] → [OpenAI] → [Chat Output]
```

### RAG 问答

```
[Chat Input] → [Vector Store] → [Prompt] → [OpenAI] → [Chat Output]
```

### 多代理系统

```
[Supervisor]
    ├── [Agent 1]
    ├── [Agent 2]
    └── [Agent 3]
```

## 高级功能

### 自定义提示词模板

使用 **Prompt** 组件创建可复用的提示词：

```
你是一个专业的{role}，请回答以下问题：

{question}
```

### 工具调用

使用 **Tool Calling** 组件实现函数调用：

1. 添加 **Tool** 组件定义工具
2. 在 OpenAI 组件中启用工具调用
3. 连接工具执行节点

### 记忆管理

使用 **Memory** 组件实现对话历史：

- **Buffer Memory**：简单缓冲
- **Window Memory**：滑动窗口
- **Summary Memory**：摘要记忆

## 导出和部署

### 导出工作流

点击 **Export** 导出为 JSON 文件。

### 部署为 API

1. 点击 **Deploy**
2. 选择 **API**
3. 复制 API 端点

```bash
curl -X POST http://localhost:7860/api/v1/run/your-flow-id \
  -H "Content-Type: application/json" \
  -d '{"input": "你好"}'
```

### 部署为聊天界面

点击 **Deploy** → **Chat Widget**，获取嵌入代码。

## 环境变量配置

可以在启动时通过环境变量配置：

```bash
export OPENAI_API_KEY="your-api-key"
export OPENAI_BASE_URL="https://api.qhaigc.net/v1"
langflow run
```

## 常见问题

**Q: 组件显示红色？**

A: 检查必填参数是否填写，API Key 是否正确。

**Q: 模型连接失败？**

A: 检查：
- Base URL 是否包含 `/v1`
- API Key 是否有效
- 网络连接是否正常

**Q: 如何查看 Token 消耗？**

A: 登录 [启航AI控制台](https://api.qhaigc.net) 查看。

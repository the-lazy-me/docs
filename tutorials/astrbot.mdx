# AstrBot 接入启航AI教程

[AstrBot](https://github.com/Soulter/AstrBot) 是多平台 AI 聊天机器人框架，支持 QQ、微信、Telegram 等。本教程介绍如何接入启航AI。

## 前置要求

- AstrBot（Python 安装）
- 启航AI API Key（[获取地址](https://api.qhaigc.net)）

## 安装 AstrBot

```bash
# 克隆仓库
git clone https://github.com/Soulter/AstrBot.git
cd AstrBot

# 安装依赖
pip install -r requirements.txt

# 启动
python main.py
```

## 配置步骤

### 1. 打开配置文件

编辑 `config.yaml` 或 `config.json`：

```yaml
# config.yaml
openai:
  api_key: "your-api-key"
  api_base: "https://api.qhaigc.net/v1"
  model: "gpt-4o"
  max_tokens: 2048
  temperature: 0.7
```

### 2. 配置多模型（可选）

```yaml
models:
  default:
    provider: openai
    api_key: "your-api-key"
    api_base: "https://api.qhaigc.net/v1"
    model: "gpt-4o"
  
  fast:
    provider: openai
    api_key: "your-api-key"
    api_base: "https://api.qhaigc.net/v1"
    model: "gpt-4o-mini"
  
  coding:
    provider: openai
    api_key: "your-api-key"
    api_base: "https://api.qhaigc.net/v1"
    model: "claude-3-5-sonnet"
```

### 3. 配置平台适配器

#### QQ 配置

```yaml
platforms:
  qq:
    type: qq
    qq: 123456789
    password: "your-password"
```

#### 微信配置

```yaml
  wechat:
    type: wechat
    token: "your-token"
```

#### Telegram 配置

```yaml
  telegram:
    type: telegram
    token: "your-bot-token"
```

### 4. 启动 AstrBot

```bash
python main.py
```

## 支持的模型

AstrBot 支持启航AI的所有模型：

| 模型 | 配置值 | 用途 |
|------|--------|------|
| GPT-4o | `gpt-4o` | 综合对话 |
| GPT-4o-mini | `gpt-4o-mini` | 快速响应 |
| Claude 3.5 Sonnet | `claude-3-5-sonnet` | 复杂任务 |
| Claude 3 Haiku | `claude-3-haiku` | 简单问答 |

## 使用方式

### 基础对话

在 QQ/微信/Telegram 中直接发送消息，AstrBot 会自动调用启航AI回复。

### 切换模型

发送命令切换模型：

```
/model gpt-4o
/model claude-3-5-sonnet
```

### 系统提示词

发送命令设置系统提示词：

```
/system 你是一个专业的编程助手
```

### 清空对话

```
/clear
```

### 查看帮助

```
/help
```

## 高级功能

### 插件开发

AstrBot 支持插件扩展：

```python
from astrbot.api import AstrBotApi

class MyPlugin:
    def __init__(self):
        self.api = AstrBotApi()
    
    def on_message(self, message):
        # 调用启航AI
        response = self.api.llm_chat(
            model="gpt-4o",
            messages=[{"role": "user", "content": message}]
        )
        return response
```

### 知识库

配置本地知识库：

```yaml
knowledge:
  enabled: true
  path: "./knowledge"
  embedding_model: "text-embedding-3-small"
```

### 多轮对话

AstrBot 默认支持多轮对话，保持上下文连贯。

## 常见问题

**Q: 连接失败？**

A: 检查：
- `api_base` 是否包含 `/v1`
- `api_key` 是否正确
- 网络能否访问 `api.qhaigc.net`

**Q: QQ 登录失败？**

A: 检查 QQ 账号和密码，或尝试扫码登录。

**Q: 如何查看 Token 消耗？**

A: 登录 [启航AI控制台](https://api.qhaigc.net) 查看。
